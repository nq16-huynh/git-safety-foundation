Hoàn thành hai lộ trình GitHub Foundations trên Microsoft Learn để nắm các kiến thức cơ bản về Git và GitHub. 
Sau đó tạo repository cá nhân và thực hành các thao tác quản lý phiên bản như commit, branch, push và pull. 
Tiếp tục làm quen với quy trình GitHub Flow thông qua việc tạo Pull Request và hợp nhất các nhánh. 
Ngoài ra, đã thực hành xử lý conflict và thực hiện rollback để hiểu cách khôi phục dự án về trạng thái ổn định khi xảy ra lỗi.
Qua quá trình học và thực hành, đã hiểu rõ hơn vai trò của Git trong việc quản lý mã nguồn và hỗ trợ làm việc với AI trong Vibe Coding.
